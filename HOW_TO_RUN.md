# كابوت – Kaboot Baloot Game

## Run on your phone in 3 steps

1. Open Terminal, go to this folder:
   ```
   cd ~/Desktop/Kaboot
   ```

2. Install packages:
   ```
   npm install
   ```

3. Start the app:
   ```
   npx expo start
   ```

4. Install **Expo Go** on your iPhone/Android, scan the QR code.

---

## Game is ready with:
- Red design matching your screenshots
- Correct bidding: حكم / سن / أشكل / بس
- One center card shown face-up during bidding
- Accurate card values (J trump=20, 9 trump=14, etc.)
- 4 players (you + AI partner vs 2 AI opponents)
- Player avatars at N/S/W/E positions
- Fanned cards at the bottom
- Score overlay after each round
- 6 arenas (الرياض → جدة → دبي → الكويت → البحرين → قطر)
